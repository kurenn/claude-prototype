# Brief: SaaS analytics dashboard

Feed this verbatim to `/prototype` (choose **Quick** mode), then score the output folder.

> Prototype a SaaS product analytics dashboard called **Pulse** for a B2B startup.
> Tone: technical, calm, confident. Inspiration: Linear and Vercel dashboards.
> Audience: a sales demo for prospective customers.
> Scope: 4 screens — Overview (KPIs + chart), Events explorer (filterable table),
> Funnels, Settings.
> Content: realistic placeholder (real-sounding metric names, event names, team members).

Expected from a good run: 4 screens, density-style layouts (compact / comfortable /
spacious), at least one empty state (e.g. a no-events-yet trial account persona),
loading + toast on the primary CTA, filter chips with skeleton reloads on the events table.
