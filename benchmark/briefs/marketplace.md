# Brief: local-services marketplace

Feed this verbatim to `/prototype` (choose **Quick** mode), then score the output folder.

> Prototype a marketplace called **Tend** for booking local home-service pros
> (cleaners, electricians, gardeners). Tone: warm, trustworthy, editorial.
> Inspiration: Airbnb and Thumbtack. Audience: a pitch to investors.
> Scope: 3 screens — Browse (search + listings), Pro detail (photos, reviews, book),
> Booking confirmation. Content: realistic placeholder.

Expected from a good run: 3 screens, photo-forward layouts (grid / gallery / list),
a buyer persona set including a new/empty state, booking CTA with loading + success
toast, reviews that read like real reviews (not lorem), every listing card clickable.
